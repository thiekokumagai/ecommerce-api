# Phase 4 Plan: 04-01 - Refatorar módulo de produtos (products)

## Objective
Refatorar o módulo de `products` para a Arquitetura Limpa Modular, separando a lógica de negócios e o controle de estoque complexo em Use Cases dedicados dentro do diretório `domain`, e mantendo controllers, repositórios Prisma e DTOs na camada de `infrastructure`.

## Tasks

### 1. Criar estrutura de pastas do domínio (products)
- [ ] Criar diretórios `src/modules/products/domain/use-cases`
- [ ] Criar diretórios `src/modules/products/domain/repositories`
- [ ] Criar diretórios `src/modules/products/domain/entities`
- [ ] Definir a interface `IProductsRepository`

### 2. Refatorar Repositório (products)
- [ ] Criar `src/modules/products/infrastructure/database/prisma-products.repository.ts`
- [ ] Implementar as operações do Prisma (listagem, criação, atualização, exclusão) utilizando a interface do domínio.

- [ ] Criar `FindProductsUseCase` (listagem).

### 4. Atualizar o Controller e DTOs (products)
